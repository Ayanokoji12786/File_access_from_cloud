# File Access From Cloud (Prototype)

A full-stack file storage and sharing prototype: upload files through a web client, store them server-side with Node.js and Express, and access them from anywhere via a shareable link.

Access it here: https://file-access-from-cloud.vercel.app/

## What it does

- Upload files through a web client
- Server-side storage and file listing
- Access uploaded files via a shareable link

## Stack

Node.js, Express, JavaScript (client + server)

## Note

This is an earlier prototype exploring cloud file access. The more complete version — with row-level security, per-user encryption, and realtime sync — is in [File_storage](https://github.com/Ayanokoji12786/File_storage).
